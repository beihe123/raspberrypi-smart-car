# raspberrypi-smart-car
基于树莓派的多功能智能小车，包含循迹、红外 / 超声波避障、红外遥控、WiFi、APP 控制全套功能
raspberrypi-smart-car/
├── docs/        硬件说明文档
├── config/      引脚配置文件
├── base_env/    任务4-6 基础环境脚本
├── hardware_driver/ 任务7-10 底层驱动
├── basic_move/  任务11-12 直行转向
├── tracking_line/ 任务13-14 循迹
├── ir_obstacle/ 任务15-16 红外避障
├── ultrasonic/  任务17-20 超声波+舵机
├── ir_remote/   任务21-22 红外遥控
├── wifi_network/任务23-25 WiFi网络
├── app_control/ 任务26 APP控制
├── main.py      项目总入口
